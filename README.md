# Malak Ait Khouya Lahcen - Portfolio

A modern, responsive portfolio built with React.js, featuring smooth animations and a custom Baby Pink theme.

## ✨ Features
- 🎨 **Baby Pink & Purple Theme** – Custom design with glassmorphism effects.
- 📱 **Responsive Design** – Fully mobile-friendly layout.
- ⚡ **Optimized Performance** – Fast loading and smooth interactions.
- 🧩 **3D Integration** – Features interactive Spline scenes.

## 🛠️ Tech Stack
- **Framework:** React.js
- **Styling:** Vanilla CSS & Bootstrap
- **Icons:** React Icons & Bootstrap Icons
- **Animation:** React Type Animation & Framer Motion

## 🚀 Getting Started
1. Clone the repository: `git clone https://github.com/malak-ak/Portfolio.git`
2. Install dependencies: `npm install`
3. Run locally: `npm start`
4. Build for production: `npm run build`
